# Global Configuration

HEMTT can optionally have a global configuration file.

## Location

| Platform | Location                                      |
|----------|-----------------------------------------------|
| Windows  | `%APPDATA%\hemtt\config.toml`                |
| Linux    | `$XDG_CONFIG_HOME/hemtt/config.toml` or `~/.config/hemtt/config.toml` |
| macOS    | `$XDG_CONFIG_HOME/hemtt/config.toml` or `~/Library/Application Support/hemtt/config.toml`
